# Logica-de-programacao-visualg
Este repositório contém 50 exercícios práticos de Lógica de Programação, desenvolvidos na linguagem VisuAlg. O objetivo deste projeto foi consolidar os fundamentos de programação, algoritmos e estruturas de dados básicas.

Tópicos abordados:

Variáveis e Tipos Primitivos

Operadores Aritméticos, Relacionais e Lógicos

Estruturas Condicionais (Se / Senão / Escolha)

Estruturas de Repetição (Enquanto / Repita / Para)

Como executar:
É necessário ter o software VisuAlg instalado. Basta baixar os arquivos .alg e abri-los no programa.
